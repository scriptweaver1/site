# Images Folder

Place your script cover images here!

## Recommended Image Specs
- **Size**: 800x450px (16:9 ratio) or 600x400px
- **Format**: JPG or PNG
- **File size**: Under 500KB for fast loading

## Usage
Reference images in your scripts.json like this:
```json
"image": "images/my-script-cover.jpg"
```

Or use external URLs:
```json
"image": "https://i.imgur.com/your-image.jpg"
```
